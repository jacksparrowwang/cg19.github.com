##解题思路简介：
1. 对于输入错误判断（最复杂）
2. 进行对应的输入创建出对应大小的数组
3. 将道路网格改为渲染迷宫网格
4. 更具输入的cell的连通在渲染网格中进行连通
5. 打印渲染网格

###测试输出结果是否正确思想

可以通过遍历迷宫，遍历是通过走迷宫的方式，通过指定一个开始点，然后通过递归的行走去遍历是否
有连通的网格，当经过一个 cell 我们记录下，再去原来输入中查找key 然后继续走，遇见下一个再通过key
去看对应的value是否对应，如果不对应或者是根本没有，这时候就可以知道自己的程序有问题
